# Notation

The canonical symbols-and-acronyms reference lives at
[`docs/NOTATION.md`](docs/NOTATION.md). It defines every operator (`⊕ ⊗ 0̄ 1̄ ∘ π
η ∞ ε`), automata symbol (`Q q₀ F Σ Γ Δ ρ`), and acronym (WFST, WFSA, CTC, RNN-T,
PDA, TN/ITN, CSR, RRWM, GCD, …) used throughout the documentation.

See also [`docs/STYLE.md`](docs/STYLE.md) for the math-notation conventions.
